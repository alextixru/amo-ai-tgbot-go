# Application Layer

Бизнес-логика бота — обработка сообщений и AI агент.

## Структура

| Пакет | Назначение |
|-------|------------|
| `ai/` | AI Agent, Tools, обработка естественного языка |
| `telegram/` | Telegram обработчики, команды, кнопки |

## Принцип работы

```
Telegram (input) → AI Agent (решает что делать) → SDK (выполняет)
```

AI Agent — главный оркестратор. Он принимает решения о том, какие операции выполнять в CRM.

## Зависимости

- Использует `domain/` (делегировано SDK)
- Использует `infrastructure/crm` для доступа к amoCRM
- Использует `infrastructure/config` для конфигурации
