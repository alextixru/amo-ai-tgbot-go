# Admin Integrations Service

Сервис для настройки интеграций, вебхуков и виджетов.

## Функции

- Управление вебхуками (Webhooks).
- Просмотр установленных виджетов (Widgets).
- Настройка кнопок на сайт и шаблонов чатов.
- Работа с короткими ссылками.

## Структура

- `service.go`: Интерфейс.
- `webhooks.go`: Вебхуки.
- `widgets.go`: Виджеты.
- `website_buttons.go`: Кнопки.
- `chat_templates.go`: Шаблоны.
- `short_links.go`: Ссылки.
