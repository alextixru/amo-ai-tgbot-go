# Customers Service

Сервис для работы с Покупателями (Customers) в amoCRM, включая транзакции и бонусную программу.

## Функции

- Поиск и управление покупателями.
- Работа с транзакциями.
- Управление бонусными баллами.
- Сегментация покупателей.

## Структура

- `service.go`: Интерфейс и инициализация.
- `customers.go`: Основные операции с покупателями.
- `bonus_points.go`: Бонусная программа.
- `transactions.go`: Транзакции.
- `segments.go`: Сегменты.
- `statuses.go`: Статусы покупателей.
