# amoCRM AI Telegram Bot

Telegram бот для управления amoCRM через естественный язык с использованием AI.

## Концепция

Пользователь общается с ботом в Telegram на естественном языке. Бот использует AI (Genkit) для понимания намерений и автоматического выполнения операций в amoCRM через SDK.

**Ключевые принципы:**
- Минимализм — только необходимый функционал
- AI-first — все операции через естественный язык
- Контекстность — AI понимает текущее состояние CRM

## Поток данных

```
Пользователь → Telegram → AI Agent → CRM Tools → amoCRM SDK → API
```

## Структура

| Директория | Назначение |
|------------|------------|
| `cmd/bot` | Точка входа приложения |
| `internal/config` | Конфигурация |
| `internal/crm` | Клиент amoCRM |
| `internal/ai` | AI агент и инструменты |
| `internal/bot` | Telegram обработчик |

## Технологии

- **Go 1.23+**
- **Genkit** — AI фреймворк от Firebase
- **go-telegram/bot** — Telegram Bot API
- **amocrm-sdk-go** — SDK для amoCRM
