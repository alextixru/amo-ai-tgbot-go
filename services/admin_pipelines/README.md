# Admin Pipelines Service

Сервис для управления воронками и статусами сделок.

## Функции

- Просмотр списка воронок.
- Управление статусами внутри воронок.
- Настройка веса и цвета статусов.

## Структура

- `service.go`: Интерфейс.
- `pipelines.go`: Управление воронками.
- `statuses.go`: Управление статусами.
