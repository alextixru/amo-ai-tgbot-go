# Activities Service

Сервис для работы с активностями, примечаниями и связями в amoCRM.

## Поддерживаемые слои (layers)

- **Tasks**: Задачи и управление ими.
- **Notes**: Примечания к сущностям.
- **Calls**: Список звонков.
- **Events**: События в CRM.
- **Files**: Файлы, прикрепленные к сущностям.
- **Links**: Связи между сущностями.
- **Tags**: Теги сущностей.
- **Subscriptions**: Подписки на события.
- **Talks**: Чаты и переписки.

## Структура

- `service.go`: Интерфейс и общая инициализация.
- `tasks.go`, `notes.go`, etc.: Логика для конкретных типов активностей.
