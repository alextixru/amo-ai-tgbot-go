# amoCRM AI Telegram Bot

## Идея проекта

Telegram бот, который позволяет управлять amoCRM через естественный язык. Пользователь пишет запрос боту — AI понимает намерение и выполняет операции в CRM.

## Ключевые принципы

- **AI-first** — все взаимодействие через естественный язык
- **Контекстный** — AI понимает текущее состояние CRM
- **Минимализм** — только необходимый функционал

## Стек

- Go 1.24+
- Genkit (AI фреймворк)
- Ollama (локальные/облачные LLM)
- go-telegram/bot
- amocrm-sdk-go

## Архитектура

```
Telegram → AI Agent (Genkit) → CRM Tools → amoCRM SDK → API
```

AI агент получает сообщение, анализирует его с учётом контекста CRM, вызывает нужные инструменты и возвращает результат.
