# cmd

Исполняемые приложения проекта.

## Концепция

Директория содержит точки входа для различных компонентов системы. Каждая поддиректория — отдельное приложение со своим `main.go`.

## Назначение

В данном проекте содержит единственное приложение — Telegram бот (`bot/`).

Здесь происходит:
- Загрузка конфигурации
- Инициализация всех компонентов
- Запуск приложения
