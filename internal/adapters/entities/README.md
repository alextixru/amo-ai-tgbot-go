# Entities Service

Сервис для работы с основными сущностями amoCRM: Сделки (Leads), Контакты (Contacts), Компании (Companies).

## Функции

- Поиск сущностей по запросу и фильтрам.
- Создание и обновление сущностей.
- Связывание сущностей между собой (Link/Unlink).

## Структура

- `service.go`: Интерфейс и общая инициализация.
- `leads.go`: Логика для сделок.
- `contacts.go`: Логика для контактов.
- `companies.go`: Логика для компаний.
