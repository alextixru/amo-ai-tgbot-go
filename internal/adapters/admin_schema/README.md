# Admin Schema Service

Сервис для управления настройками и схемой данных amoCRM (кастомные поля, группы полей, причины отказа и т.д.).

## Функции

- Управление кастомными полями (Custom Fields).
- Настройка групп полей.
- Управление причинами отказа (Loss Reasons).
- Настройка источников сделок (Sources).

## Структура

- `service.go`: Интерфейс.
- `custom_fields.go`: Кастомные поля.
- `loss_reasons.go`: Причины отказа.
- `sources.go`: Источники.
