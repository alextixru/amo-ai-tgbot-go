# ai

AI агент и инструменты.

## Концепция

Мозг приложения. Использует Genkit для обработки естественного языка и принятия решений о действиях в CRM.

## Назначение

- Инициализация Genkit с выбранной моделью
- Определение Tools (инструментов) для операций с CRM
- Обработка пользовательских сообщений
- Формирование контекста из данных CRM

## Компоненты

| Файл | Назначение |
|------|------------|
| `agent.go` | AI агент, оркестрация обработки |
| `tools.go` | Genkit Tools для CRM операций |

## Принцип работы

AI агент получает сообщение пользователя, анализирует его с учётом контекста CRM, и вызывает соответствующие инструменты для выполнения операций.
