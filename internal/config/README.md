# config

Конфигурация приложения.

## Концепция

Централизованное управление настройками приложения. Все параметры загружаются из переменных окружения при старте.

## Назначение

- Определение структуры конфигурации
- Загрузка параметров из ENV
- Валидация обязательных параметров

## Параметры

| Переменная | Назначение |
|------------|------------|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота |
| `AMOCRM_BASE_URL` | URL аккаунта amoCRM |
| `AMOCRM_ACCESS_TOKEN` | Токен доступа к API |
| `GENKIT_MODEL` | Модель AI |
