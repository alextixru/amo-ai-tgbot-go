# bot

Telegram интеграция.

## Концепция

Транспортный слой приложения. Отвечает за получение сообщений из Telegram и отправку ответов пользователю.

## Назначение

- Обработка входящих сообщений от пользователей
- Передача текста в AI агент
- Отправка ответа обратно в Telegram
- Обработка ошибок и edge cases

## Принцип работы

Получает Update от Telegram API, извлекает текст сообщения, передаёт его AI агенту, получает ответ и отправляет пользователю.

## Ответственность

Этот пакет не содержит бизнес-логики. Он лишь связывает Telegram с AI агентом.
