# internal

Внутренние пакеты приложения.

## Концепция

Содержит всю бизнес-логику приложения, разделённую на независимые компоненты. Пакеты в `internal` не могут быть импортированы извне проекта.

## Структура

| Пакет | Ответственность |
|-------|-----------------|
| `config` | Конфигурация приложения |
| `crm` | Взаимодействие с amoCRM |
| `ai` | AI обработка и инструменты |
| `bot` | Telegram интеграция |

## Зависимости между пакетами

```
bot → ai → crm → config
```

Каждый пакет зависит только от нижележащих, что обеспечивает чистую архитектуру.
