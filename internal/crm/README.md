# crm

Клиент amoCRM.

## Концепция

Обёртка над amoCRM SDK, предоставляющая упрощённый интерфейс для AI инструментов. Изолирует детали работы с API от остальной логики.

## Назначение

- Инициализация SDK с credentials
- Методы-обёртки для операций с CRM
- Формирование AI-контекста через `ai.GlobalState`
- Преобразование данных между SDK и AI слоем

## Ответственность

Этот пакет отвечает только за взаимодействие с amoCRM API. Он не знает ни о Telegram, ни о специфике AI.
